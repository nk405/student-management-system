Courses

Foundation

  • Introduction to fundamental principles and concepts of software engineering.
  • Focus on building a strong foundation in programming and software development methodologies.
  • Covers essential programming languages like Java and Python.
  • Hands-on experience in developing basic software applications.
  • Preparatory course for further studies or entry-level positions in software engineering.

HND

  • Comprehensive two-year program for in-depth study of software engineering.
  • Builds upon foundational knowledge with advanced programming languages, software architecture, and database management.
  • Emphasizes real-world applications, projects, and practical skills.
  • Enables graduates to work as software engineers or software developers.
  • Pathway for further studies in the field.

Degree

  • Rigorous and in-depth program spanning three to four years.
  • Covers software development, system analysis, project management, and quality assurance.
  • Advanced study of programming languages, software design principles, algorithms, and data structures.
  • Emphasis on practical experience through internships or industry projects.
  • Prepares graduates for leadership roles in the software engineering industry or advanced studies.
